## 🎛️ Using ROBLOX-MACRO-V3.0.0

### Creating Your First Macro

1. Open the application.  
2. Click **New Macro** or **Record** to begin capturing your input sequence.  
3. Press the keys or mouse buttons in the order you want the macro to repeat.  
4. Click **Stop Recording** when finished.  
5. Review the list of recorded inputs. You can delete or rearrange steps if needed.  
6. Save the macro with a clear name for easy access later.

### Running a Macro

1. Select the macro from the list.  
2. Click **Start** to begin running the input sequence automatically.  
3. The macro will send inputs as you recorded them.  
4. Click **Stop** when finished or to pause the sequence.

### Common Controls Explained

- **Delay**: Sets how long to wait between each input. Helps control speed.  
- **Repeat Count**: Number of times the macro cycles through all inputs.  
- **Loop Mode**: Set the macro to repeat indefinitely until manually stopped.  
- **Hotkeys**: Assign a keyboard shortcut to start or stop macros quickly.
